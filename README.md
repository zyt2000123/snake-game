# 贪吃蛇游戏

一个使用HTML5 Canvas和JavaScript实现的经典贪吃蛇游戏。

## 功能特点

- 🎮 支持键盘和按钮控制
- 📱 响应式设计，支持移动设备
- 🏆 本地最高分记录
- ⏸️ 暂停/继续功能
- 🎨 现代化UI设计

## 游戏说明

- 使用方向键或点击按钮控制蛇的移动
- 吃到食物可以增长身体和得分
- 避免撞到墙壁或自己的身体
- 按空格键可以暂停/继续游戏

## 在线体验

直接打开 `index.html` 文件即可开始游戏！

## 技术栈

- HTML5 Canvas
- JavaScript (ES6+)
- CSS3 (渐变、动画、响应式)
